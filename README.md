# Koshisan Home Assistant Add-ons

Custom Home Assistant Add-ons by koshisan.

## Installation

1. Open Home Assistant
2. Go to **Settings** → **Add-ons** → **Add-on Store**
3. Click the **⋮** menu (top right) → **Repositories**
4. Add this repository URL:
   ```
   https://github.com/koshisan/homeassistant-addons
   ```
5. Refresh the Add-on Store
6. Install the desired add-on

## Add-ons

### Wyoming OpenAI

OpenAI-Compatible TTS/STT bridge for Wyoming Protocol with text preprocessing.

**Features:**
- Automatic emoji and markdown removal from TTS text
- Custom text replacements for better pronunciation
- Works with tts-webui, LocalAI, OpenAI, Groq, and more
- Based on [Wyoming OpenAI](https://github.com/roryeckel/wyoming_openai)

[More info →](wyoming-openai/README.md)
